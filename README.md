# 好好照顾自己 🌿

一个轻量级的**行为触发型健康管理工具**，帮助你养成良好的生活习惯，建立自律坚持的力量。

## ✨ 核心特性

### 🎯 智能生活锚点触发
- **不是时间提醒，而是行为触发** — 基于生活节奏（早餐后、午餐后、晚餐后、睡前）自动唤醒
- 用户自我标记行为 → 系统主动推荐 → 快速完成任务
- 避免打扰，让健康管理融入自然生活

### 💊 三层计划管理
1. **长期坚持** — 每天必做，建立连续打卡
2. **短期计划** — 设定天数，自动到期停用
3. **按需使用** — 记录症状，追踪用药原因

### 📊 深度数据洞察
- **时间范围筛选** — 7天/30天/90天/全部查看
- **身体状态趋势** — 可视化心情数据，平均评分一目了然
- **症状统计排行** — 最常见症状 Top 5 分析
- **药物使用频率** — 哪个药最常用一目了然
- **历史记录查询** — 最近50条记录精确到分钟

### 🔔 多层提醒体系
- **轻提醒** — 整点温柔toast提示，不打扰但能唤醒
- **智能弹窗** — 在正确时间段自动推送相关任务
- **重新访问检查** — 页面切换时自动检测未完成任务

### 📅 完整日历与记录管理
- 生活周期视图 — 星期标签直观显示
- 日期粒度记录 — 每条记录精确到分钟
- 选中日期详情 — 查看特定日期的所有记录和身体状态

## 💾 数据存储

所有数据保存在浏览器 `localStorage` 中，**永不联网**：

```javascript
{
  medications: [],      // 计划列表
  records: [],         // 使用记录
  moods: [],          // 心情/身体状态记录
  streak: {},         // 连续打卡信息
  settings: {}        // 用户设置
}
```

**数据持久化** — 刷新页面后数据仍保留  
**隐私优先** — 完全离线，无任何云端同步

```
   → 症状是否减少了

5. 月底导出数据

   → 数据说话，医学决策更科学
```

## 💡 核心算法

### 智能触发逻辑
```
IF 当前时间在 [早餐时段:6-10点]
  AND 有未完成的"早餐后"药物
  AND 今天还没弹过这个时段
  THEN 自动打开早餐弹窗
```

优势：
- 减少打扰
- 增加及时性
- 一天每个时段最多弹一次

### 连续打卡计算
- 检查前一天是否有记录
- 若有 → 连续天数+1
- 若无 → 重置为0
- 考虑"跨天阈值"（凌晨算前一天）

### 完成率公式
```
完成率 = 有记录的天数 / 开始记录以来的总天数 × 100%
```

## 📞 使用建议

### 最佳实践
1. **立即开始** — 不要等待"完美时机"，现在就添加一个计划
2. **坚持记录** — 即使一天只喝了一杯水，也要记录
3. **周末复盘** — 看看这周的趋势，调整策略
4. **导出分享** — 定期导出给医生，医学决策更有据可依
5. **设置提醒** — 打开通知，让系统温柔地引导你

### 常见问题

**Q: 数据会丢失吗？**  
A: 不会。数据保存在浏览器 localStorage，只要不清除浏览器数据就永久保留。

**Q: 能多个设备同步吗？**  
A: 目前不能。每个设备独立存储。可通过导出CSV在不同设备手动导入。

**Q: 离线可以用吗？**  
A: 完全可以。第一次打开后自动缓存，此后无网络也能完全使用。

**Q: 如何备份数据？**  
A: 设置页面 → 数据管理 → 导出为CSV，保存到云盘即可。
